## 1. Τα retrieval metrics είναι λανθασμένα

Η εργασία απαιτεί:

 Recall@K
 MRR
 nDCG@K

### Προβλήματα:

 Το Recall@K υπολογίζεται ως `correct / k` (λάθος)

       def recallk_score(predictions, correct_results, k):
           if len(correct_results) == 0:
               return 0.0
           correct_guess = 0
           for prediction in predictions:
               if prediction in correct_results:
                   correct_guess += 1
       
           return correct_guess/k

 Το RR/MRR δεν βασίζεται στη θέση του πρώτου relevant στο ranking

          def RR_score(predictions, correct_results):
          for indx, answer in enumerate(correct_results):
              if answer in predictions:
                  return 1/(indx + 1)
      
          return 0

 Το nDCG δεν έχει σωστό relevance formulation

      def DCGk_score(predictions, correct_results, k):
    gt_scores = {id: len(correct_results) - i for i, id in enumerate(correct_results)}

    true_relevance = np.array([[gt_scores.get(id, 0) for id in predictions]])
    predicted_scores = np.array([[len(predictions) - i for i in range(len(predictions))]])

    return ndcg_score(true_relevance, predicted_scores, k=k)

### Τι πρέπει να κάνεις:

 Recall@K:

```text
(# relevant στο top-K) / (# συνολικά relevant)
```

 RR:

```text
1 / θέση πρώτου relevant στο ranking
```

Αν δεν διορθωθούν αυτά, τα αποτελέσματα δεν είναι αξιόπιστα.

## 2. Τα edge weights στο kNN graph είναι λάθος

Στο kNN graph χρησιμοποιείς cosine distance:

```text
distance = 1 - similarity
G.add_edge(node_id, ids[nearest], weight=distances[i][j])
```

Αλλά ο semantic graph πρέπει να βασίζεται σε:

```text
similarity (cosine)
```

### Τι πρέπει να κάνεις:

```python
sim = 1.0 - distance
G.add_edge(u, v, weight=sim)
```

Το βάρος της ακμής πρέπει να εκφράζει semantic closeness, όχι απόσταση.

## Phase 3 — Graph-based retrieval (core της εργασίας)

### Στόχος

Να χρησιμοποιήσεις τον γράφο για βελτίωση retrieval.

---

### 3.1 Personalized PageRank (PPR)

#### Λογική:

```text
1. Υπολόγισε cosine(query, docs)
2. Πάρε initial relevance scores
3. Τρέξε PageRank με personalization vector = query scores
4. Πάρε νέο ranking
```

---

### 3.2 k-step neighborhood expansion

#### Λογική:

```text
1. Πάρε top-K docs από cosine
2. Βρες neighbors στο graph (1-hop, 2-hop)
3. Δώσε score με βάση:
   - similarity
   - distance στο graph
4. Φτιάξε νέο ranking
```

---

### 3.3 Fusion (πολύ σημαντικό)

Συνδύασε:

```text
final_score = α * cosine + (1 - α) * graph_score
```

Δοκίμασε π.χ.:

```text
α = 0.3, 0.5, 0.7
```
