# Semantic_graph_construction_and_graph_based_retrieval
## Περιγραφή 
```text
Αυτό το project υλοποιεί ένα σύστημα ανάκτησης πληροφορίας (Information Retrieval System) 
που δημιουργεί σημασιολογικούς γράφους χρησιμοποιώντας διαφορετικές μεθόδους αναπαράστασης και ανάλυσης κειμένου.
Στόχος είναι η σύγκριση της αποτελεσματικότητας των παραγόμενων σημασιολογικών γράφων με την κλασική μέθοδο ομοιότητας συνημιτόνου (Cosine Similarity) για την εύρεση της βέλτιστης προσέγγισης στην ανάκτηση σχετικών πληροφοριών.
```
### Features
```text
- Παραγωγή σημασιολογικών αναπαραστάσεων με χρήση του μοντέλου SBERT (Sentence-BERT).
- Κατασκευή σημασιολογικών γράφων με διαφορετικές μεθόδους, όπως k-NN, Mutual k-NN, Clustering κ.ά.
- Ανάκτηση πληροφορίας από τους γράφους με τεχνικές όπως Personalized PageRank (PPR) και K-step Neighborhood Expansion.
- Χρήση reranking τεχνικών για τη βελτίωση των αποτελεσμάτων ανάκτησης.
- Αξιολόγηση και σύγκριση των αποτελεσμάτων των διαφορετικών μεθόδων με μετρικές αξιολόγισης recall@k , MRR, Ndcg@k, MAP@k.
```

## Οδηγίες
Κατά την εκτέλεση του προγράμματος εμφανίζεται το παρακάτω μενού:

- Make a test graph
- Run retrieval
- Make embeddings
- Make leaderboard

### Make a Test Graph

Η επιλογή Make a Test Graph εμφανίζει ένα επιπλέον μενού με τις διαθέσιμες μεθόδους κατασκευής γράφων. Στη συνέχεια, ο χρήστης καλείται να εισαγάγει τις απαραίτητες παραμέτρους για τη δημιουργία του γράφου (π.χ. αριθμό γειτόνων)

### Run retrieval
Η επιλογή Run retrieval τρέχει για όλους τους προκατασκευασμένους γράφους όλες τις μεθόδους ανάκτησης πληροφορίας για διαφορετικές παραμέτρους

### Make embeddings
Η επιλογή Make embeddings παράγει τα embeddings με την χρήση του SBERT μοντέλου και τα αποθηκεύει στο αρχείο embeddings.pkl. Παράλληλα δημιουργεί ένα αρχείο texts.pkl που περιέχει ολα τα κείμενα απο τα passages αντιστοιχισμένα με τα id τους για το reranking

### Make leaderboard
Η επιλογή Make leaderboard δημιουργεί ένα global leaderboard μεταξύ όλων των γράφων (δεν έχει γίνει ακόμα)
